# 🦇 Nocturna — Готическая эстетика на Vuetify

Небольшой проект, демонстрирующий возможности фреймворка Vuetify в рамках тематики «готическая эстетика».

## 🎯 Цель проекта

Создать атмосферное веб-приложение, которое использует ключевые компоненты Vuetify:
- **Лейаут** (App Bar, Navigation Drawer, Footer)
- **Контейнеры и сетка** (Grid System)
- **Карточки** (v-card, v-img)
- **Диалоги** (v-dialog)
- **Уведомления** (v-snackbar)
- **Навигация** (v-btn, v-parallax)

## 🛠️ Используемые технологии

- Vue.js 2
- Vuetify 2.7.4
- Material Design Icons (MDI)
- Google Fonts (Cormorant Garamond)
